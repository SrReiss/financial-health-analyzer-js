# financial-health-analyzer-js
Script em JavaScript que analisa faturamento, custos, lucro e margem de empresas, classificando a saúde financeira e gerando relatório automatizado.

# Análise de Lucratividade de Empresas

Projeto em JavaScript que simula um diagnóstico financeiro automatizado para pequenas empresas, calculando lucro, margem e classificação de saúde financeira.

Este projeto foi desenvolvido como prática de lógica de programação aplicada à análise contábil e consultoria financeira.

---

# Funcionalidades

- Filtra empresas ativas
- Calcula lucro e margem de cada empresa
- Classifica a saúde financeira:
  - Crítica
  - Atenção
  - Saudável
- Formata valores no padrão monetário brasileiro
- Gera relatório automático no console
